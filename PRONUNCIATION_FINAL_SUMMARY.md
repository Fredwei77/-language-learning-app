## 🔧 技术实现

### 数据结构
```typescript
interface PracticeItem {
  id: string
  text: string
  phonetic: string
  translation: string
  difficulty: "easy" | "medium" | "hard"
  category: string
  type: "word" | "phrase" | "sentence" | "passage"
  audioUrl?: string
  tips?: string[]
}
```

### 核心函数
- `getAllPracticeItems()` - 获取所有练习
- `getPracticeItemsByType()` - 按类型筛选
- `getPracticeItemsByDifficulty()` - 按难度筛选
- `getPracticeItemsByCategory()` - 按分类筛选
- `getRandomPracticeItem()` - 随机获取（支持筛选）

### 状态管理
- 练习类型状态
- 难度筛选状态
- 分类筛选状态
- 可用题目列表
- 当前题目状态
