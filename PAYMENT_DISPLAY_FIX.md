# 💳 支付表单显示问题 - 快速修复

## 🔴 问题
支付表单底部被截断，用户看不到完整的支付按钮。

## ✅ 推荐解决方案：切换到全屏模式

### 为什么选择全屏模式？
- ✅ 内容100%完整显示
- ✅ 无滚动问题
- ✅ 移动端体验优秀
- ✅ 实现简单可靠

---

## 🚀 快速修复（1分钟）

### 步骤 1: 打开文件
打开 `app/coins/page.tsx`

### 步骤 2: 找到导入语句
找到这一行（大约在第 20 行）：
```typescript
import { PaymentDialog } from "@/components/payment-dialog"
```

### 步骤 3: 替换为全屏模式
替换为：
```typescript
import { PaymentDialogFullscreen as PaymentDialog } from "@/components/payment-dialog-fullscreen"
```

### 步骤 4: 保存文件
保存后，页面会自动刷新（热重载）

### 步骤 5: 测试
访问 http://localhost:3000/coins，点击任意套餐，支付表单现在应该全屏显示！
