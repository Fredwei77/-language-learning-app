## 🎓 使用方法

### 1. 选择练习
- 使用顶部筛选器选择类型、难度、主题
- 或点击"随机题目"快速开始

### 2. 听示范
- 点击"听示范"按钮
- 仔细听标准发音
- 阅读发音技巧提示

### 3. 开始录音
- 点击麦克风按钮
- 清晰朗读内容
- 再次点击停止

### 4. 查看结果
- AI自动评测
- 查看综合得分
- 阅读改进建议

## 🆘 常见问题

**Q: 麦克风无法使用？**

**Q: 识别不准确？**
