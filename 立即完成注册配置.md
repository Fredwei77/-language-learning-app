## 📝 需要做的事（2分钟）

### 在 Supabase Dashboard 中添加回调 URL

1. **保持当前的 Supabase 页面打开**
   
2. **在 Redirect URLs 部分，点击 "Add URL" 按钮**

3. **输入以下 URL：**
   ```
   https://good2study.netlify.app/auth/callback
   ```

4. **点击 "Save changes" 保存**

5. **等待 10 秒让配置生效**

## ✅ 配置完成后的样子

你的 Redirect URLs 应该显示：

```
✓ https://good2study.netlify.app/
✓ https://good2study.netlify.app/auth/callback  ← 新添加的
```

## 🧪 然后立即测试

### 快速测试步骤：

1. **访问注册页面**
   ```
   https://good2study.netlify.app/auth/register
   ```

2. **填写信息并注册**
   - 邮箱：使用真实邮箱
   - 密码：至少6个字符

3. **检查邮箱**
   - 查找 Supabase 发来的验证邮件
   - 可能在垃圾邮件文件夹

4. **点击验证链接**
   - 应该看到 "验证成功" 页面
   - 3秒后自动跳转到登录页

5. **登录测试**
   - 用刚注册的账号登录
   - 应该能成功进入网站
